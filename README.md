# 📊 Data Analysis Basics with Excel

## 📌 Project Overview

This project demonstrates basic data analysis techniques using Microsoft Excel. The dataset contains over **32,000 Data Analyst job postings** from 2023 and is used to perform simple data cleaning, summarization, and statistical analysis.

The objective of this project is to practice fundamental Excel skills that are commonly used in data analysis.

---

## 📂 Dataset

The dataset contains the following columns:

- ID
- Date
- Job Title
- Job Location
- Hired
- Easy Apply
- Month

A separate **Summary** worksheet contains monthly application statistics.

---

## 📈 Analysis Performed

The project includes:

- Organizing raw data
- Monthly application summary
- Total applications
- Minimum applications
- Maximum applications
- Average applications
- Basic statistical calculations using Excel formulas

---

## 🛠 Tools Used

- Microsoft Excel
- Excel Functions
  - SUM()
  - MIN()
  - MAX()
  - AVERAGE()
- Basic spreadsheet formatting
